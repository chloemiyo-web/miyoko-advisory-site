# Miyoko Advisory — Studio site

A boutique modernization studio for established businesses.
Next.js 14 (App Router), Tailwind CSS, deployed on Vercel.

## Local development

```bash
npm install
npm run dev
```

Visit http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## Structure

- `app/` — Pages: Home, Services, Work, Apply, Privacy, Terms
- `components/` — Nav, Footer, Section, ApplyForm, FadeUp
- `app/globals.css` — Color tokens, typography, components
